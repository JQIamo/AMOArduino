typedef void(* GenericSetListCallback)(void *, int *)

Type-agnostic callback function for SetList devices.

This defines a generic callback function type. Since we don't know a priori what device types are controllable by the Arduino, this casts the device type to void *. Later, when the callback function is inserted into SetListDevice::_setlist, we re-cast the void * pointer to the proper type.

There is perhaps a more elegant way to do this, but the current structure seems to work nicely.
